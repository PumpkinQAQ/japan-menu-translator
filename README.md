# 🇯🇵 日本旅遊菜單翻譯點餐助手

使用 Gemini AI 翻譯日本餐廳菜單，幫你輕鬆點餐！

## ✨ 功能

- 📸 拍照或上傳菜單圖片
- 🤖 AI 自動翻譯日文菜單
- 🛒 互動式選擇餐點
- 💴 自動計算總金額（日幣/台幣）

## 🚀 部署

1. 前往 [Streamlit Cloud](https://share.streamlit.io)
2. 用 GitHub 登入
3. 點 **New App** → 選擇此 Repo
4. 點 **Deploy!**

## 🔑 取得 API Key

前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 取得 Gemini API Key

## 📱 使用方式

1. 輸入 API Key
2. 拍照或上傳菜單
3. 點擊翻譯
4. 選擇餐點
5. 查看總金額

## 💡 提示

- 建議使用 Gemini API Key（免費版即可）
- 支援日文、英文等語言
- 在日本旅遊時隨開即用！
